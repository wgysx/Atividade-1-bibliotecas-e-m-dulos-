# Repositório de Exercícios Python

Este repositório contém os exercícios solicitados:

## Exercícios

1. **Pedra, Papel e Tesoura** (`pedra_papel_tesoura.py`)
   - Simula o jogo Jo Ken Po usando o módulo random

2. **MegaSena** (`megasena.py`)
   - Gera 6 números únicos de 1 a 60 para a MegaSena

3. **Calculadora** (`calculadora.py` + `main.py`)
   - Módulo com operações matemáticas
   - Importação e uso no arquivo principal

4. **Cowsay** (`cowsay_exemplo.py`)
   - Exemplos de uso da biblioteca cowsay

5. **Cowsay Personalizado** (`desenho_personalizado.py`) - *Opcional (+0.25)*
   - Desenhos personalizados usando a função .draw()
